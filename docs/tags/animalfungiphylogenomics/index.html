CGM 第125期：重建动物和真菌的生命树

2020-11-25
重建早期动物的系统进化关系是进化生物学研究的热点和难点问题之一。这一难题对我们理解动物进化的早期事件（包括神经系统的起源）具有重要意义。一些研究支持海绵动物，另一些研究支持栉水母动物为最早的动物进化分支。这些分析在很多方面都有所不同，包括所基因和物种的选择，分子进化模型和软件的不同。在这里，我们通过综合15项以前的研究的数据和结果，并进行一套全面的新的标准化分析，系统地探索了动物生命树的最早进化分……
