CGM 第149期: 单细胞谱系追踪解析免疫系统的发育

2021-04-14
多细胞生命是从单个细胞开始，通过细胞分裂和分化，最终形成组织分明、井然有序的个体。在这个过程中细胞的发育历史被称为谱系 (lineage)。记录、追踪并绘制细胞发育历史的过程被称为谱系追踪 (lineage tracing) 。近年来科学家通过发展谱系追踪的方法学，深入理解了干细胞驱动的器官再生与稳态，推动了体外迷你器官培养的进步。在癌症领域，谱系追踪揭示了肿瘤细胞的起源和克隆演化，提供了靶向肿瘤……
